# NtVP Workshop - Novel Methods to Evaluate Trauma Treatments in Research and Care: Bayesian and IC Evaluation of Informative Hypotheses

Welcome! This folder contains all the materials needed for our workshop session at the NtVP conference on May 29, 2026 in Utrecht. You do not need any programming knowledge to use these files; we will be using JASP, an open-source graphical software for statistical analysis.

### What’s in this folder?

* NtVP_slides.pdf: The slides we will follow during the workshop.
* patient1_data.csv: The raw (synthetic) dataset we will be analyzing during the workshop.
* patient2_data.csv: An additional raw (synthetic) dataset to practice with. It will be discussed during the workshop if time allows.
* patient1_jasp.jasp & patient2_jasp.jasp: These files contain the completed analyses. You can open these in JASP to see the results and the steps we took. We recommend first trying to run the analyses on your own.
* "Sexual revictimisation risk mitigation among survivors of sexual violence and trafficking": a PDF of the article mentioned in the presentation and used as an example.

### How to download these files

If you are not familiar with GitHub, follow these steps to get the files onto your computer:

* Download Everything: Go to the main repository for the BETTR project https://github.com/gorizzon/BETTR. Look toward the top right and click the green "<> Code" button.
* Select Download ZIP: Click "Download ZIP". This will download the entire project as a folder to your computer.
* Unzip: Open the downloaded file. All the BETTR files are now on your computer. Inside the BETTR folder you can find the NtVP Workshop subfolder. We suggest moving the NtVP Workshop folder to your desktop.
* Note: Once you have moved the NtVP Workshop subfolder, you can delete the BETTR folder and the remaining files inside - they are not needed for the analyses. If you only want a specific file (for example, the presentation), click on the file name in the list above, then click the Download raw file icon (it looks like a small downward arrow) on the right side of the screen.

### Software Required

To open the .jasp files, please ensure you have downloaded and installed JASP (which is free) from jasp-stats.org.
